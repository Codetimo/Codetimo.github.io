# Codetimo AI Frontier 中文收藏夹

这个 GitHub Pages 项目现在用于收藏、翻译与分享 OpenAI、Anthropic 的最新进展、深度文章和 Claude Code 学习资料。

## 当前定位

- 面向中文读者的 AI 前沿资料库
- 优先收录 OpenAI、Anthropic、Claude 官方来源
- 重点整理模型更新、产品进展、Coding Agent、创业者文章与视频教程
- 后续可扩展为多篇中文翻译与笔记页面

## 首批栏目

- 精选收藏：重点文章和教程入口
- OpenAI 观察：OpenAI News、Codex、API 文档
- Anthropic 观察：Anthropic Newsroom、Claude、Claude Code
- Claude Code 中文学习路线：官方文档、视频课程、进阶主题

## 维护建议

后续可以把每篇文章拆成独立页面，例如：

```text
/articles/
  founders-playbook-ai-native-startup.html
  claude-code-101-notes.html
  codex-long-running-work.html
```

也可以增加一个 `data/resources.json` 来集中维护链接、标签、摘要和翻译状态。
